# Game Timer

**Приложение:** https://egmen.github.io/gamer/

Таймер хода для настольных игр. Простое веб-приложение, которое отсчитывает
время хода игрока и подаёт звуковые сигналы о старте хода, приближении конца
времени и его истечении. Удобно для многопользовательских настольных игр, где
нужно ограничивать время на ход.

## Возможности

- **Отсчёт времени хода.** Нажатие на экран запускает таймер заново. Время хода
  показано кольцом точек (1 точка = 1 секунда): прошедшие секунды закрашиваются,
  в центре — счётчик оставшихся секунд.
- **Световые сигналы.** Фон вспыхивает на старте хода, в зоне предупреждения и
  по окончании времени.
- **Звуковые сигналы** генерируются на лету через Web Audio API (без сетевых
  запросов и аудиофайлов). Доступные наборы:
  - **ЧГК** — гармонические тоны в духе «Что? Где? Когда?» (спектр снят с
    оригинальных записей и воспроизводится через `PeriodicWave`), по умолчанию;
  - **Стандарт** — простые бипы из исходного дизайна;
  - **Выкл** — звук отключён.
- **Темы оформления:** Полночь, Уголь, Индиго.
- **Настройки** сохраняются в `localStorage` и переживают перезагрузку страницы:
  - время хода (`moveTime`, по умолчанию 60 секунд);
  - время предупреждения (`warningTime`, по умолчанию 10 секунд до конца);
  - тема (`theme`) и набор звуков (`soundSet`);
  - кнопка «Сбросить по умолчанию» возвращает исходные настройки.
- **Wake Lock** — экран не гаснет, пока открыто приложение (на поддерживаемых
  устройствах), что удобно для использования с телефона или планшета на столе.

## Как пользоваться

1. Откройте приложение в браузере.
2. Нажмите кнопку **«Меню»** в правом верхнем углу — снизу выедет панель
   настроек, где можно задать время хода и предупреждения, выбрать тему и набор
   звуков.
3. Нажмите на область таймера, чтобы (пере)запустить отсчёт хода. По истечении
   времени прозвучит сигнал.

## Технологии

- [React 19](https://react.dev/) + TypeScript
- [MobX](https://mobx.js.org/) для управления состоянием
- [Vite](https://vitejs.dev/) для сборки и dev-сервера
- [pnpm](https://pnpm.io/) как пакетный менеджер
- Web Audio API для генерации звуков
- Screen Wake Lock API

## Структура проекта

```
src/
├── index.tsx                # точка входа, монтирование React, wake lock
├── settings.ts              # стор настроек с сохранением в localStorage
├── timer.ts                 # стор таймера (MobX): отсчёт, фазы, сигналы
├── themes.ts                # темы оформления (цветовые палитры)
├── wakeLock.ts              # запрос Screen Wake Lock
├── screen/
│   ├── App.tsx              # корневой компонент-обёртка
│   ├── Timer.tsx            # экран таймера: кольцо точек, меню, панель настроек
│   ├── view.ts              # геометрия точек по кругу и оверлей-вспышки фона
│   ├── ui.ts                # переиспользуемые inline-стили
│   └── components/
│       ├── DotRing.tsx          # кольцо точек со счётчиком в центре
│       ├── SettingsSheet.tsx    # выезжающая снизу панель настроек
│       ├── SegmentedControl.tsx # переключатель темы / набора звуков
│       └── Stepper.tsx          # шаговый ввод (−/+) времени
└── sound/
    └── playAudio.ts         # генерация звуковых сигналов через Web Audio API
```

## Запуск

```bash
pnpm install     # установка зависимостей
pnpm dev         # запуск dev-сервера (http://localhost:3000)
pnpm build       # production-сборка в папку build/
pnpm preview     # локальный просмотр собранной версии
pnpm lint        # проверка кода ESLint
```
